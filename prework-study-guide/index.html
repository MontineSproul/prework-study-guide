Prework Study Guide

HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • The paragraph element will contain most of the text for a webpage.
  • For an image tag it must contain a src attribute, this will be where the image is stored.
    • It is best practice to add an alt attribute as well. This is useful if the image does not load or an assistant is being used. The alt attribute is a description of the picture.
    • The image tage is a self closing tag which means it uses only one tag for the element.
  • There are six different heading elements h1, h2, h3, h4, h5 and h6. h1 should only be used once per page.
  • There are two different types of list elements, an unordered list and an ordered list. Each of these elements will have the child element of li.

CSS

  • Cascading Style sheet or CSS is how you add style to your webpage.
  • There are three different ways to add CSS: inline, external and internal.
  • Inorder to see your CSS external file reflected on the webpage it needs to be linked to the HTML.
  • You can style multiple elements at the same time by seperating them with a comma.
  • * is a wildcard selector. There are a few different selectors such as by element or by class attribute.
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside of an element.
